# Social Media Image Sizes Cheat Sheet 2026

Every platform has different image dimensions. Posting the wrong size means blurry photos, awkward cropping, or wasted screen space. Here is the complete reference for every major platform in 2026.

## Instagram Image Sizes

| Format | Size | Aspect Ratio |
|--------|------|-------------|
| Square post | 1080 × 1080 | 1:1 |
| Portrait post | 1080 × 1350 | 4:5 |
| Landscape post | 1080 × 566 | 1.91:1 |
| Story / Reels | 1080 × 1920 | 9:16 |
| Reel cover | 1080 × 1920 | 9:16 |
| Profile picture | 320 × 320 | 1:1 |
| Carousel | 1080 × 1080 or 1080 × 1350 | 1:1 or 4:5 |

**Tip:** The 4:5 portrait format (1080 × 1350) gets 17-30% more engagement than square posts because it takes up more screen space in the feed.

## TikTok Image Sizes

| Format | Size | Aspect Ratio |
|--------|------|-------------|
| Photo post | 1080 × 1920 | 9:16 |
| Video cover | 1080 × 1920 | 9:16 |
| Profile picture | 200 × 200 | 1:1 |

TikTok is entirely vertical-first. All image content displays in 9:16 format.

## Facebook Image Sizes

| Format | Size | Aspect Ratio |
|--------|------|-------------|
| Post (landscape) | 1200 × 630 | 1.91:1 |
| Post (square) | 1080 × 1080 | 1:1 |
| Story | 1080 × 1920 | 9:16 |
| Cover photo | 820 × 312 | 2.63:1 |
| Profile picture | 170 × 170 | 1:1 |
| Event cover | 1920 × 1005 | 1.91:1 |
| Group cover | 1640 × 924 | 1.79:1 |

## YouTube Image Sizes

| Format | Size | Aspect Ratio |
|--------|------|-------------|
| Thumbnail | 1280 × 720 | 16:9 |
| Channel banner | 2560 × 1440 | 16:9 |
| Profile picture | 800 × 800 | 1:1 |
| Community post | 1200 × 675 | 16:9 |

**Tip:** YouTube thumbnails are the single most important image for getting clicks. Always use 1280 × 720 with high contrast and readable text.

## LinkedIn Image Sizes

| Format | Size | Aspect Ratio |
|--------|------|-------------|
| Post (landscape) | 1200 × 627 | 1.91:1 |
| Post (square) | 1080 × 1080 | 1:1 |
| Profile picture | 400 × 400 | 1:1 |
| Banner | 1584 × 396 | 4:1 |
| Company logo | 300 × 300 | 1:1 |
| Newsletter cover | 600 × 600 | 1:1 |

**Tip:** LinkedIn profile photos that are at least 400 × 400 pixels appear sharp on all devices. Blurry LinkedIn photos are a common complaint.

## Pinterest Image Sizes

| Format | Size | Aspect Ratio |
|--------|------|-------------|
| Standard pin | 1000 × 1500 | 2:3 |
| Square pin | 1000 × 1000 | 1:1 |
| Story pin | 1080 × 1920 | 9:16 |
| Profile picture | 165 × 165 | 1:1 |

**Tip:** Pinterest strongly favors 2:3 vertical pins (1000 × 1500). Horizontal pins get significantly less distribution.

## X / Twitter Image Sizes

| Format | Size | Aspect Ratio |
|--------|------|-------------|
| Post (single image) | 1600 × 900 | 16:9 |
| Post (4-image grid) | 1200 × 675 each | 16:9 |
| Header | 1500 × 500 | 3:1 |
| Profile picture | 400 × 400 | 1:1 |
| Summary card | 1200 × 628 | 1.91:1 |
